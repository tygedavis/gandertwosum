# GanderSum

## Instructions for local instalation
* Clone this repository
* In one terminal, cd into /app and run `npm run start`
* In another terminal, cd into /api and run `npm run serve`
* Add all the numbers you want


### API Documentation
```
POST /addTwoNums

{
    num1: Integer
    num2: Integer
}
```
